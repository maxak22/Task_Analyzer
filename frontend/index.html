JSON Format:

  • title (required): Task name
  • due_date (optional): YYYY-MM-DD format
  • estimated_hours (optional): Number of hours
  • importance (optional): 1-10 scale
  • dependencies (optional): Array of task IDs

Limits:

  • Maximum 100 tasks per import
  • Title is required for each task
  • Importance must be 1-10
  • Dependencies will be validated
